지금 까지 배운 html 태그들

  • html
    문서 작성시 맨처음 사용한다. 상위에 !doctype html을 같이 써줘야한다.
  • head
    본문(body)를 설명해주는 역할이다 즉 데이터 방식,제목 등을 쓴다.
  • title
    웹페이지의 이름을 설정하는 태그이다.
  • meta
    브라우저에서 해당 웹을 불러올때 어떤 형식으로 불러올지를 결정해준다 보통 utf-8로 설정한다.
  • body
    본문을 작성하기전에 쓰는 태그이다. 이태그를 쓰고 그밑에 내용을 쓴다.
  • h1~h6
    본문의 제목 태그라고 생각하면된다.
  • p
    본문내용 태그이다.
  • br
    라인을 바꿔주는 태그이다.
  • u
    언더라인 즉 밑줄을 만들어준는 태그이다.
  • strong
    해당 태그 안의 내용들을 굵게 표시해준다.
  • li
    리스트태그로써 단독으로 사용할 수 있으나 그것보다는 ol이나 ul태그와 같이 사용하는 것이 좋다.
  • ol
    리스트태그로써 각 리스트마다 앞에 번호를 붙여준다.
  • ul
    리스트태그로써 ol태그와 다르게 번호를 붙여주지 않고 구분만 시켜준다.
  • img
    src로 이미지 파일의 위치를 지정해서 width로 크기를 설정해서 웹에 보여준다.
    파일의 확장자명까지 정확하게 써야한다.
  • table
    표를 만드는 태그인데 tr과 td와 함께 써야한다.
  • tr
    테이블의 세로 열을 만드는 태그이다.
  • td
    테이블의 가로 줄을 만드는 태그이다.
  • a
    링크를 걸어주는 태그이다.
    새 탭으로 띄우려면 target="_blank"를 추가하고 툴팁을 추가하려면 title="내용"을 추가하면된다.
